
## 模块 1️⃣：损失模型（Loss Model）

### 你要做什么

- 模拟 **一年内的保险损失**

### 最简但专业的做法

1. **事故频率**：Poisson
2. **事故损失大小**：Lognormal / Pareto

```text
一年内事故数 N ~ Poisson(λ)
每次事故损失 X ~ Lognormal(μ, σ)
```


---

## 模块 2️⃣：再保险合同函数（最关键）

### 你必须写一个清晰的函数：

```python
def reinsurance_payout(loss, retention, limit):
    return min(max(loss - retention, 0), limit)
```

### 你要解释清楚

- 再保险不是“分摊”
- 是 **非线性截断函数**

## 模块 5️⃣：定价逻辑（不需要太复杂）

你只需要一个 **非常真实的再保险公式**：

```text
Reinsurance Premium
= Expected Loss
× (1 + Risk Loading)
+ Expenses
```

👉 Risk Loading 可以来自：

- TVaR – EL
- 或一个简单系数（比如 20%）
