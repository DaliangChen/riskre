# 📌 再保险定价、风险与资本建模平台（原型系统）

**作者：** Daliang Chen  
**地点：** 德国 · 法兰克福

---

## 1. 项目目标与范围

本项目是一个**面向业务的再保险分析平台原型**，用于模拟和展示在非寿险再保险环境中，
**定价、风险评估与资本建模**是如何在统一框架下协同运作的。

系统围绕**超赔再保险（Excess of Loss, XoL）**结构，构建了一条从
**历史数据与建模假设 → 风险建模 → 定价指标 → 资本影响 → 管理层报告**
的**可复现、可审计、情景驱动的完整工作流**。

> 本项目的目标不是“演示一个仿真程序”，
> 而是展示在真实业务环境中，
> **精算思维、风险建模与软件工程是如何结合，支持承保与管理决策的。**

---

## 2. 业务背景

在非寿险再保险业务中，组合表现与合约定价主要受到以下因素驱动：

- 损失频率与损失严重度的不确定性
- 尾部风险与资本占用
- 通胀、气候等宏观环境变化
- 承保组合与暴露结构的变化

本系统旨在支持：

- **承保人员（Underwriters）**：条款结构与价格决策
- **风险与资本团队（Risk & Capital）**：TVaR、SCR 代理指标与情景冲击分析
- **管理层（Management）**：风险调整后收益与压力测试结果解读

## 3. 核心功能

### 3.1 数据与治理

- 历史赔案与暴露数据导入与校验
- 建模假设与情景参数基于 YAML 文件进行版本化管理
- 模型运行的可复现性（数据、参数与模型快照存档）

### 3.2 建模模块

- **频率模型**：泊松分布、负二项分布
- **严重度模型**：经验分布、对数正态、GPD / 帕累托、体尾拼接模型
- **依赖结构（可选）**：Copula / 共同冲击因子模型

### 3.3 再保险结构建模

- 超赔结构（单层 / 多层）
- 起赔点与限额分析
- 层损失曲线与赔付模拟

### 3.4 风险与资本指标

- 期望损失（EL）
- 在险价值（VaR）
- 尾部在险价值（TVaR）
- 可能最大损失（PML）
- SCR 代理指标 / 经济资本（基于情景分析）

### 3.5 情景与压力测试

- 赔款通胀冲击
- 暴露增长与组合漂移
- 气候与灾害强度因子
- 多情景对比与指标变化分析

### 3.6 报告输出

- 自动生成**承保定价报告（PDF）**
- 风险委员会摘要报告
- 情景对比表格与图表

## 4. 系统架构

```text
data/           # 历史赔案、暴露数据、外部指数
config/         # 基础假设与情景配置（YAML）
models/         # 频率、严重度、依赖结构与拟合逻辑
contracts/      # 超赔与多层再保险结构
pricing/        # 保费、风险加载与费用模型
risk/           # VaR、TVaR、SCR代理与资本指标
scenarios/      # 情景引擎与压力测试
reporting/      # PDF与管理层输出
runs/           # 模型运行快照与结果归档
```

## 5. 工作流程

```text
1. 数据导入与质量校验
2. 建模假设与情景选择
3. 模型参数估计与拟合诊断
4. 年度总损失模拟
5. 再保险层结构应用
6. 定价与资本指标计算
7. 情景对比分析
8. 承保与管理层报告输出
9. 运行记录与审计追踪
```

## 6. 建模理念

本平台采用**分层、渐进式建模方法**：

- 基础参数模型用于透明性与可解释性
- 数据充分区域使用经验或半参数方法
- 针对超赔层与资本分析引入重尾与体尾拼接模型
- 通过情景驱动方式进行前瞻性风险评估

该方法反映了在**Solvency II / ORSA 框架下**，
再保险公司内部模型与定价系统的常见实践。

---

## 7. 假设与模型治理

所有关键建模与定价假设均通过配置文件进行管理，包括：

- 分布类型选择
- 尾部分界点（Threshold）
- 风险加载方法
- 通胀与气候参数
- 资本置信水平

每一次模型运行都会生成：

- 唯一运行编号（Run ID）
- 数据版本引用
- 参数快照
- 模型版本记录

用于支持审计、复现与模型风险管理。
